Module 15 Final Project

This website was created for the Web Development 280 Final Project. The client for this project was GreenTech Solutions, an environmental sustainability and technology company.

HTML

Semantic HTML5 elements were used to structure the website, including navigation, sections, articles, forms, headings, lists, and paragraphs.

CSS

External CSS styling was used to create a responsive, Earth Day inspired eco technology theme with accessibility improvements and mobile responsiveness.

JavaScript

JavaScript was used to create product search filtering and contact form validation for user interaction.

Accessibility Features

  • Semantic HTML structure
  • Alt text for images
  • Keyboard focus indicators
  • Responsive mobile design
  • Accessible forms with labels
  • High-contrast color palette
